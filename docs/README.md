## Grammar
``` nemet
// Identifiers
{ _ | a-Z }{ _ | a-Z | 0-9 }*

// Entry point
fun main() {}

// declearing a variable
// static variable
let PI :: 3.14;
// const veriable
let buffer_len := 12;
// mutable variable
let a = 5;
a = a + 1;

// while loops
while (condition) {}

// if statments
if (condition1) {}
else if (condition2) {}
else {}
```
